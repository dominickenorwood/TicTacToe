TIC-TAC-TOE.

1. Developed for Chrome but should work for FireFox and Safari.

2. First player that begins is X.

3. Winner declared if player gets a match horizontally, vertically or at a diagonal.

4. Reset game with 'Rest Game' button.

5. To run this app:
    - Run "npm install" in command line
    - Run "npm run start" to start servers on localhost:8080
    - Run "npm run test" to initialize unit tests.

6. Alternatively you can open "index.html" in the "dist" folder in your local browser.
